# Credit Scoring Model

## Objective
Predict whether a loan applicant is creditworthy using historical financial data.

## Dataset
German Credit Dataset (1000 records, 20 features)

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest achieved:
- Accuracy: 75%
- Recall: 84%
- ROC-AUC: 0.79

## Techniques
- Feature scaling and one-hot encoding
- Class imbalance handled using SMOTE
- Model evaluation using Precision, Recall, F1-score, ROC-AUC
